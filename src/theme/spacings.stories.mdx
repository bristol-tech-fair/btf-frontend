# Spacings

Predefined spacings keep our sizings consistent through out the application. Please only ever use the spacings defined below.

<br />
<br />

### How to use spacings

When creating components you'll want to use these spacings in your code.

Adding them in to your `Component.styles.js` file is simple, click the 'copy' button beneath the space value.

And paste it where you would normally add your pixel or rem value.

```js
import styled from 'styled-components';

export const ExampleWrapper = styled.div`
  padding: var(--small-space);
`;
```
