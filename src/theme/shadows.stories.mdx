# Palette

A predefined color palette helps to ensure brand consistency across the application. Please only use the colors defined below.

<br />
<br />

### How to use the palette

When creating components you'll want to use these colors in your code.

Adding them in to your `Component.styles.js` file is simple, click the 'copy' button beneath the color.

And paste it where you would normally add your hexcode or rgb value.

```js
import styled from 'styled-components';

export const ExampleWrapper = styled.div`
  border: 0.1rem solid var(--logopink);
`;
```
