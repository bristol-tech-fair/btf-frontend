# Shadows

A predefined set of box shadows helps to ensure brand consistency across the application. Please only use the shadows defined below.

<br />
<br />

### How to use the shadows

When creating components you'll want to use these shadows in your code.

Adding them in to your `Component.styles.js` file is simple, click the 'copy' button beneath the shadow.

And paste it where you would normally add your `box-shadow` value.

```js
import styled from 'styled-components';

export const ExampleWrapper = styled.div`
  box-shadow: var(--card-shadow);
`;
```
